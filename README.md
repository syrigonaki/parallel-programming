### Parallel Programming

A collection of implementations focused on concurrency and synchronization.


### Projects:
* Game of Life (OpenMP): Parallel grid updates using shared-memory data parallelism and double-buffering.
  
* Elevator System (Akka/Scala): Concurrent multi-agent simulation using the Actor model and asynchronous message passing.
  
* N-Body Simulation (Intel TBB): Implementation of the Barnes-Hut algorithm using spatial partitioning.
  
* PageRank (Pthreads): Multi-threaded graph processing and iterative rank computation using POSIX threads.
  
* Bus Simulation (Semaphores): Low-level thread synchronization and resource management using Pthreads and semaphores.
